#+TITLE: Static And Build Output

* static

=static/= is copied through as public files.

Use it for assets that should have stable public paths, such as:

- favicon
- robots.txt
- images
- screenshots

* build

=build/= is generated by =yarn build=.

It is the final static site produced by SvelteKit and =adapter-static=. Do not
edit it by hand.

* .svelte-kit

=.svelte-kit/= is generated by SvelteKit. It contains temporary build output
and generated TypeScript files used by =svelte-check=.

Do not edit it by hand either.
