* What It Is

Personal site for =nexenne.com=: portfolio, technical writing, work case
studies, resume, contact page, command palette, bilingual routes, RSS, sitemap,
Open Graph images, and PWA/offline support.

It is built with SvelteKit, generated as static HTML, and fed by local Markdown
and Org content. Production is GitHub Pages serving the generated =build/=
folder.

* Quick Start

#+begin_src sh
yarn install --immutable
yarn dev
yarn check
yarn test
yarn build
#+end_src

The project uses [[https://yarnpkg.com/][Yarn 4]], pinned via =packageManager= in
=package.json= and =.yarnrc.yml=. You only need a Yarn launcher on =PATH=; the
repo's checked-in Yarn release does the rest.

* Quality Gates

Run these before pushing a change:

#+begin_src sh
yarn format:check
yarn check
yarn lint
yarn test
yarn build
#+end_src

GitHub Actions mirrors the same idea:

- [[file:.github/workflows/quality.yml][quality.yml]]: format, Svelte/TypeScript check, lint, tests, build.
- [[file:.github/workflows/lighthouse.yml][lighthouse.yml]]: Lighthouse budgets against built pages.
- [[file:.github/workflows/static.yaml][static.yaml]]: GitHub Pages deploy from =build/=.

* Environment

=PUBLIC_WEB3FORMS_ACCESS_KEY= enables contact-form delivery through Web3Forms.
Without it, the contact form falls back to =mailto:=. Copy =.env.example= to
=.env= for local testing and set the same name as a GitHub Actions secret for
production.
